# Tic-Tac-Toe AI

A full-stack Tic-Tac-Toe web application featuring an unbeatable AI agent. This project implements the **Minimax Algorithm** with **Alpha-Beta Pruning** to ensure the AI never loses.

## Features
*   **Unbeatable AI:** Uses game theory to evaluate every possible move.
*   **Optimized Search:** Alpha-Beta pruning reduces the number of nodes evaluated by the algorithm, making it highly efficient.
*   **Modern Web UI:** Clean, responsive interface built with HTML/CSS.
*   **Cloud Ready:** Configured for instant deployment on Vercel.

## Tech Stack
*   **Backend:** Python (Flask)
*   **Frontend:** HTML5, CSS3, JavaScript (ES6)
*   **Deployment:** Vercel (Serverless Functions)
